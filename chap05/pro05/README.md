# 5장 서블릿 이해하기 예제 프로젝트 - pro05

> * eclipse를 사용하지 않아서 설정면에서 책하고 차이가 나는 예제가 되겠지만. 목표는 서블릿 클래스 구현이므로 Gradle + Gretty 프로젝트로 만들어도 되겠다.
> * 이번에는 JUnit이나 Spring Test의 MockHttpServletRequest/Respose로 테스트가 가능한지 확인해봐야겠다. 



## 실행 방법

```bash
> gradle clean appRun
```

이후 브라우저에서 아래 주소로 접속

* `http://localhost:8080/`
